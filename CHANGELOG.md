# Changelog - Candidex AI

All notable changes to the Candidex AI Career Platform will be documented in this file.

## [1.0.0] - 2026-07-04

### Added
- **AI Career Agent:** Conversational dashboard chatbot using intent classifiers.
- **RAG Knowledge Base:** Pickle-backed lightweight vector database and text chunker.
- **AI Research Lab:** Evaluates prompt configurations, latencies, and intent precision.
- **Job Copilot:** Computes application readiness scores and outlines tailoring steps.
- **ML Intelligence:** SentenceTransformers integration for cosine similarity.
- **Observability Layer:** Centralized JSONL request metrics and user feedback loggers.
