# ATS Optimization Guidelines

1. **Quantifiable Metrics:** Always quantify achievements. Instead of writing "Responsible for server configuration," write "Configured server groups, boosting load delivery speeds by 24%."
2. **Standard Section Names:** Use clear headers like "Education," "Experience," "Projects," and "Skills." Avoid fancy labels.
3. **No Embedded Graphics:** Avoid tables or text boxes, as old ATS parser scripts skip them during imports.
4. **Keyword Alignment:** Mirror critical keywords listed directly under the target Job Description to rank high on match algorithms.
